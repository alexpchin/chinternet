Hello.js and Rails

Sep 16, 2014

I thought I’d have a little play with integrating hello.js and Rails. Devise and Oauth2 are quite good but I liked the idea of doing all of this functionality client-side.

    1. Create a new rails app (I’m using Rails 4.)
    1. Add relavent gems to Gemfile and bundle
    1. Scaffold basic User and install Devise
    1. Update development.rb by adding:
    1. Change initializer for Devise.
    1. Add Devise to User model
    1. Run migrations
    1. Bower install Hello.js (I use this setup for Bower & Rails)
    1. Register your app domain, let’s do it on Facebook
    1. Include hello.all.js script in application.js (hello.all includes all of the provider modules)
    1. Create the signin buttons in application.html.erb. I used the zocial to style.
    1. Setup listener for login and retrieve user info.
    1. Initiate the client_ids and all listeners (Client ID is aka App ID)
    1. Add root route
    1. Create a new user using Devise using the information from returned from hello.js for a provider
